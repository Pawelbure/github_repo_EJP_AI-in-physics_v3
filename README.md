# An Instructional Proposal for Teaching AI Integration in Physics

The entire code and datasets for the manuscript can be found in this repository. Please be aware that the code is inspired by the listed references cited on top of each notebook.

You find instruction sheets in the folder ``1_instruction-sheets''. These can be read first. They are intended as instruction materials to implement the activities. 

The folders are then named after the respective use case chapters in the mansucript.

You can run the notebooks via Binder (see link below). It is recommended to run the notebooks on your own PC, given the restricted compute resources in Binder's container. You need to have Python installed, as well as Jupyter lab or Jupyter Notebook. The file ``requirements.txt'' lists the required Python modules.

Note: Results in your computations might differ from the exemplary use cases in the paper.
